Notes on Brownian Motion

Steven R. Dunbar, Ph.D.

sdunbarne@gmail.com

Notes on a subject that fascinates me, namely Brownian Motion, a stochastic process with continuous sample paths which are non-differentiable almost everywhere. These notes are written for mathematicians, using graduate level mathematical analysis and probability theory. The format of the notes is a style I use for distributing class notes to students with a Starter Question for discussion, a list of Key Concepts, a list of defintions of new Vocabulary, a list of Notations to help keep notation consistent across notes, the main section fo Mathematical Ideas, some computer Scripts and some references and bibliography entries.
